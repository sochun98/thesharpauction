# CLAUDE.md

Claude Code가 이 프로젝트에서 따라야 할 행동 규칙을 기록합니다.

---

## Q&A 기록 규칙

대화 중 사용자가 질문하고 Claude가 답변한 내용은 **반드시 `docs/QUERY.md`에 누적 저장**합니다.

### 저장 형식

```markdown
## YYYY-MM-DD

**Q. 질문 내용**

A. 답변 내용
```

### 적용 시점

- 사용자가 기술적 질문을 하고 Claude가 조사·분석 후 답변할 때마다 저장
- 대화가 끝나기 전, 또는 다음 작업으로 넘어가기 전에 저장
- 이미 저장된 내용과 중복되지 않도록 날짜 + 질문으로 구분
